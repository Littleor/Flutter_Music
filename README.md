## 简介 
借用[网易云API](https://github.com/Binaryify/NeteaseCloudMusicApi)开发的一个Flutter下载音乐应用.

## 使用 
```bash
git clone https://github.com/Littleor/Flutter_Music
cd Flutter_Music
flutter pub get
flutter run
#flutter build apk #打包APK
```
